Samszone E-Commerce Company Online Inventory System:

Please select an action in the following form, then provide parameter(s) when appropriate.

If you choose to 'List Items', you may specify the range as start,end as a pair in the parameter box. If you enter nothing into the Parameter box, the whole inventory will be printed.

If you choose 'Search by keyword', enter a text pattern or keyword below in the Parameter box. The search is case sensitive.

If you select 'Value of the Inventory', leave the Parameter box blank to return the dollar value of the whole inventory.

If you select 'Search by Category', enter the category you wish to see. The following are category options to look through:

  • Books
  • CD Vinyls
  • Collectibles
  • Electronics
  • Fashion
  • Home Garden
